
## DATA FLOW VERIFICATION

### Complete Working Flow:
1. NestJS controller receives HTTP request
2. Service method called (e.g., getRecommendationsAI)
3. Redis cache check
4. HTTP POST to Rust with:
   - URL: http://localhost:8000/endpoint
   - Headers: { "x-api-key": "..." }
   - Body: JSON payload
   - Timeout: 5000ms
   - Retry: 2 attempts
5. Rust middleware validates API key
6. Handler processes request
7. Database queries executed (PostgreSQL)
8. Response serialized to JSON
9. Backend receives response
10. Data saved to database (learning_suggestion table)
11. Redis cache updated
12. Response returned to client
