# STUDY-GROUPS Module

## Overview

This module provides functionality for study-groups features.

## Structure

- components/ - React components
- hooks/ - Custom React hooks
- services/ - API service layer
- types/ - TypeScript type definitions
- utils/ - Utility functions

## Getting Started

1. Review the types in types/
2. Implement API services in services/
3. Create React hooks in hooks/
4. Build UI components in components/

## API Integration

See services/index.ts for available API methods.

## Testing

Run tests with: npm test -- study-groups
